# stryve

> A real-time communication platform.

## Build Setup

``` bash
# install dependencies
yarn

# serve with hot reload at localhost:9080
yarn dev

# build electron app for production
yarn build

# build web app for production
yarn build:web

# lint all JS/Vue
yarn lint

# run unit tests
yarn unit

# run e2e tests
yarn e2e

# run all tests
yarn test
```
